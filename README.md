# meteor-blaze-react-component

Use Blaze templates inside of React.

```jsx
import React from 'react';
import Blaze from 'meteor/blaze-react-component';

const App = () => (
  <div>
    <Blaze template="itemsList" items={items} />
  </div>
);
```
